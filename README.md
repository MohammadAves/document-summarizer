# Document Summarization Service

## Description
This project is a simple document summarization service using OpenAI GPT API. It supports different summarization styles: brief, detailed, and bullet points. Users can input text via a web interface.

## Features
- Input text via web interface
- Multiple summarization styles
- Handles API errors gracefully
- Basic input validation
